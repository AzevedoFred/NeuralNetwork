1. Criando o ambiente virtual

Nessa primeira etapa você já deve ter o Anaconda Python 3.6 instalado. Se não, basta fazer o download do instalador nesse link. A instalação é bem simples bastando clicar Next várias vezes.
Agora, com o Anaconda Python 3.6 instalado, abra o Anaconda Navigator e clique na aba Environments. Em seguida clique no botão Create. No campo Name digite um nome para seu ambiente virtual, evite espaços, acentos e caracteres especiais enquanto estiver escolhendo o nome. Eu escolhi MyVenv. Por fim clique em Create.
Pronto, seu ambiente virtual foi criado!!!

2. Ativando seu ambiente virtual

Abra o Anaconda Prompt no seu Windows. Se estiver com dificuldade para encontrá-lo, utilize a busca do próprio Windows. Ao abrir o prompt você estará com uma tela como a abaixo.

Recomendo deixar seus projetos dentro do drive C para facilitar a busca. O projeto que uparemos para o Github se encontra na pasta MyProject dentro do drive C.
